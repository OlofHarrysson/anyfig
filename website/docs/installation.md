## Requirements
Python 3.7+

## Installation
The Anyfig package is in its infancy. Install it from pip or the github repo

```bash
# Latest "stable" realease
pip install anyfig

# From github
pip install git+https://github.com/OlofHarrysson/anyfig/archive/master.zip

```
